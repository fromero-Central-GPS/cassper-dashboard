# Email Reactivation Agent (GOG / GHL Tags)

This script manages sending reactivation emails to lost leads using the GOG email provider or by applying specific tags in GHL to trigger a native workflow.

## Phase 2 Goals
1. Connect to GOG (or configure GHL tags) for email sending.
2. Read the daily prioritized report (generated by recovery-mvp).
3. Draft a tailored reactivation email based on Claude's analysis.
4. Update GHL custom fields/tags to record the contact attempt.
